API Interface Description

URL return type returns description
http://ip:port/events/..
 ..events/update bool true if update was successful transmit FireEvent via POST in Body
 ../all FireEvent[] Array of all FireEvents
 ../source/{sourceId} FireEvent[] returns all FireEvent from one FireAlarmSystem
 ../id/{sourceId}/{eventId} FireEvent returns a specific FireEvent
 ../target/{sourceId}/{targetId} FireEvent[] returns all FireEvents of a distinct target of a FireAlarmSystem
 ../type/{sourceId}/{eventType} FireEvent[] returns all FireEvents from a FireAlarmSystem with the given EventType
 ../type/{eventType} FireEvent[] returns all FireEvents with the given EventType
 ../time/{sourceId}/{startTime}/{endTime} FireEvent[] returns a list of all FireEvents with a matching sourceId and and a Timestamp between startTime and endTime
 ../time/{startTime}/{endTime} FireEvent[] returns a list of all FireEvents with a Timestamp between startTime and endTime
 ../date/{sourceId}/{year}/{month}/{day} FireEvent[] returns a list of all FireEvents of a FireAlarmSystem on the given date
 ../date/{sourceId}/{year}/{month} FireEvent[] returns a list of all FireEvents of a FireAlarmSystem in the given year and month
 ../date/{sourceId}/{year} FireEvent[] returns a list of all FireEvents of a FireAlarmSystem in the given year
 ../typeyear/{eventType}/{year} Int32[12] returns an array with the number of FireEvents of the given EventType that occured in the given year.
Each column represents one month
